# Fredricks BV Indicator Primers

## Targets (borrowed from Fredricks et al. 2007)

These 16S-based species-specific primers fill gaps where cpn60 cannot reach.

| Target | Why include | BV diagnostic value |
|--------|-----------|-------------------|
| BVAB1 | Still uncultured; not in cpnDB; strongly associated with inflammation | High |
| BVAB2 (Acetatifactor indicium) | Best single BV biomarker | Sensitivity 98.8% (combined with Megasphaera) |
| Megasphaera type 1 | Distinguished from type 2 only by specific PCR; linked to preterm birth | Specificity 93.7% (combined with BVAB2) |
| Fannyhessea vaginae | Key BV indicator; not well-resolved by cpn60 | High |

## Source
Fredricks, D.N. et al. (2007). JCM, 45, 3270-3276. DOI: 10.1128/JCM.01272-07
- Table 1 contains all 17 species-specific primer sequences
- PMC free text: https://pmc.ncbi.nlm.nih.gov/articles/PMC2045326/

## Notes
- All primers target 16S rRNA variable regions (species-specific)
- Validated on 264 vaginal fluid samples (81 BV+, 183 BV-)
- Designed for US population — in silico validation against SA 16S sequences recommended before use
