# cpn60 Database

## Sources
- **cpn60-Classifier v11.1**: 16,413 curated sequences — [GitHub](https://github.com/HillLabSask/cpn60-Classifier)
- **cpnDB_nr_vag**: Vaginal-specific reference set — contact Hill lab or cpndb.ca
- **VOGUE Reference Assembly**: OTUs from 546 vaginal microbiomes — contact Hill lab

## Setup
```bash
# Clone the cpn60-Classifier repository
git clone https://github.com/HillLabSask/cpn60-Classifier.git

# Training sets are in the releases
# v11.1 contains: trained classifiers, QIIME2 artifacts, raw sequence/taxonomy files
```

## Key reference
Ren, Q. & Hill, J.E. (2023). ISME Communications, 3, 77. DOI: 10.1038/s43705-023-00283-z
